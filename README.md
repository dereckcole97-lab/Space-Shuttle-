# Space-Shuttle
## Description 
Implementing an interactive Space Shuttle departure simulation showcasing mission status updates in real time
## Features
- Interactive control: the launch is controlled by clicking a button
- Flight simulation: real-time animation showing the shuttle movement towards to top
- Visual indicator: the starting of a countdown before the departure
## Technologies 
- HTML5
- CSS3
- JAVASCRIPT
